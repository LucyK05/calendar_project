# Calendar App Information

## Purpose
The purpose of this app is to provide the user with a functional calendar where they can add their events and tasks with corresponding times and color coding

## Functions
**Functional Calendar Layout** - The user is able to navigate through calendar to select the exact month and day they are looking for
**Current Date Tracker** - The app highlights the current day on the calendar to keep the user up-to-date
**Adding Events** - The user can click the add event button on any day and choose the name, time frame (if applicable), and color for each
**Event Deletion Button** - The user can delete any added event from the calendar using the provided button

## AI Use
The code for the calendar app was built using Grok generative AI. The AI wrote the index.html and app.py files after being fed multiple prompts to change the event adding mechanism and calendar theme.
